# Changelog

## [1.0.0] - 2025-11-02

### Added
- Initial release of FinPilot MVP
- Django REST API backend with PostgreSQL
- Next.js frontend with authentication
- ML service (FastAPI) with PyTorch models for expense classification and report generation
- Celery background jobs for transaction processing
- Stripe billing integration (test mode)
- Plaid sandbox/mock integration for bank connections
- Docker Compose setup for local development
- Seed script for demo data
- ML dashboard for training metrics visualization

### Features
- User registration and JWT authentication
- Organization management
- Transaction ingestion and classification
- 90-day cashflow forecasting
- Anomaly detection
- AI-generated weekly reports
- Account connection management
- Billing and subscription management
